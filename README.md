This project provides a complete workflow for running and analyzing a synchronized EEG and eye-tracking experiment using
  Mentalab Explore Pro, Pupil Labs Neon, and PsychoPy.

  1. Experiment Execution
  File: psychopy_script_Mentalab_Neon.py  
  This is the core PsychoPy script for running the experiment.
   - Task Design: Implements an active/passive trial structure (40 trials each). In the "active" condition, participants
     press a key to hear a tone; in the "passive" condition, the tone plays after a random delay.
   - Data Streaming: Streams task events (markers) via LSL (Lab Streaming Layer) as the PSY_MARKERS_TASK stream.
   - Device Integration: Connects to the Pupil Labs Neon Real-Time API to send synchronized events and control recording.
     It also integrates with Mentalab Explore for EEG data collection via LSL.
   - Synchronization: Handles PC-to-Neon clock offset estimation to ensure precisely timed event markers.

  2. Data Analysis
  File: analysis_notebook.ipynb  
  A Jupyter notebook designed to process the .xdf files recorded by Lab Recorder.
   - Data Loading: Uses pyxdf to load EEG, Pupil Diameter data, and Task Marker streams.
   - Preprocessing: Resamples eye-tracking data to match the EEG sampling rate and optionally filters EEG data.
   - MNE Integration: Creates an MNE-Python RawArray containing both EEG (e.g., Fz, FCz, Cz, CPz) and pupil diameter (Left,
     Right) channels.
   - Event Annotation: Converts the LSL markers into MNE annotations for trial-based analysis.

  3. Results Visualization
  File: synced_event_plot.png  
  This is the visual output generated by the analysis notebook.
   - Unified View: Shows synchronized EEG and pupil diameter traces on a single timeline for the first few seconds of the experiment.
   - Event Mapping: Features vertical dashed markers and labels (e.g., sound_onset_active, sound_onset_passive) that
     precisely align with the physiological signals, verifying the temporal accuracy of the multi-modal recording.
